Exercicio 1

Crie uma página HTML com dois campos de entrada para um número. Ao clicar em um botão e realize a soma.Faça a mesma coisa para subtração, multiplicação e divisão.

Exercicio 2

Crie uma página HTML com um campo de entrada para um número. Ao clicar em um botão, verifique se o número é par ou ímpar e exiba uma mensagem correspondente.

Exercicio 3

Crie uma página HTML com um campo de entrada para um número. Ao clicar em um botão, exiba a tabuada desse número de 1 a 10.

Exercicio 4

Crie uma página HTML com campos de entrada para três notas. Ao clicar em um botão, calcule a média dessas notas e exiba se o aluno foi aprovado ou reprovado (média >= 7 para aprovação).

Exercicio 5

Crie uma página HTML com um campo de entrada para um número. Ao clicar em um botão, exiba todos os números pares de 0 até o número inserido.

Exercicio 6

Crie uma página HTML com três campos de entrada para números. Ao clicar em um botão, verifique qual é o maior dos três números e exiba-o.

Exercicio 7

Crie uma página HTML que solicite um número ao usuário. Em seguida, exiba todos os números ímpares de 1 até o número inserido usando um loop while.

Exercicio 8

Crie uma página HTML com um jogo de adivinhação simples. O computador deve gerar um número aleatório de 1 a 100, e o jogador deve tentar adivinhar o número. O jogo deve dar feedback se o número é maior ou menor, até o jogador acertar.

Exercicio 9

Crie uma página HTML com um menu para escolher entre operações de soma, subtração, multiplicação e divisão. Em seguida, exiba campos de entrada para os números e o resultado da operação selecionada.
